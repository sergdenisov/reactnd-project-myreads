The final assessment project for Udacity's React Fundamentals course, developed by [React Training](https://reacttraining.com).

## How to start

```bash
git clone git@github.com:sergdenisov/reactnd-project-myreads.git
cd reactnd-project-myreads
npm install
npm start
```

To see the project, open `http://localhost:3000` in your browser.

## Components documentation

This project uses [React Styleguidist](https://github.com/styleguidist/react-styleguidist), so after installing you can
start a documentation server by:

```bash
npm run styleguide
```

To see the documentation, open `http://0.0.0.0:6060` in your browser.
